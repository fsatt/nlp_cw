# NLP Coursework: Patronising Language Detection

This repository contains the code, data analysis, and evaluation files for the NLP coursework on detecting Patronising and Condescending Language (PCL). This corresponds to SemEval 2022 Task 4 (Subtask 1).

## Repository Structure
- `BestModel/BestModel.ipynb` : Source code for the final submitted model.
- `RoBERTa_baseline.ipynb` : Baseline used for comparison (BestModel minus the proposed approach).
- `EDA.ipynb` : Exploratory Data Analysis.
- `dev.txt` & `test.txt` : The final binary predictions generated by the BestModel for global evaluation.

*Note: All notebooks are designed to run out-of-the-box and will automatically download the necessary datasets when executed.*

## Running the Fine-Tuned Model
I provide an already fine-tuned model that you can download and run without needing to retrain from scratch.

You can download the zip file manually from 
[this Google Drive link](https://drive.google.com/file/d/1J2igfeLy1H2EbTHTqtn7j5hA9X4osTWy/view?usp=drive_link) 
and extract it into the `BestModel/` directory, or simply run the following commands:

```bash
# Download the model zip
wget 'https://drive.google.com/uc?export=download&id=1J2igfeLy1H2EbTHTqtn7j5hA9X4osTWy' -O BestModel/saved_model.zip

# Unzip into BestModel/
unzip BestModel/saved_model.zip -d BestModel/
```

Then, open `BestModel/BestModel.ipynb` and set the `USE_SAVED_MODEL` variable in  to `True`.

Finally, run all cells to evaluate the model and generate predictions.
